Problems from an Interview


1. Approximating a biased coin as fair

First, given a biased coin, come up with a solution to approximate a fair coin using the biased coin.

For e.g., for a coin with 0.7 chance of a heads, the solution to this would be to flip the coin twice. And if the flips result in two heads, we accept that as a single Heads as:

P(HH) = 0.7 * 0.7 = 0.49

and we use all the other permutations as a proxy for the Tails approximation:

P(HT) + P(TH) + P(TT) = 2 * (0.3 * 0.7) + (0.3 * 0.3) = 0.51
